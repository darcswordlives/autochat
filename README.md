# Autochat

An extension for SillyTavern that sends a user message after a random amount of time.

## Features

- Set a minimum and maximum number of seconds.
- Customize the message to be sent.
- Enable/disable the timer with a single checkbox.

## Installation

### Method 1: Local Install

1. Navigate to `SillyTavern/public/scripts/extensions/third-party/`.
2. Create a new folder named `autochat`.
3. Place the provided files inside this folder.
4. Refresh SillyTavern. The extension will appear in the Extensions settings.

### Method 2: GitHub Install

1. Upload these files to a new GitHub repository.
2. In SillyTavern, go to `Extensions` -> `Install Extension`.
3. Paste the GitHub URL of your repository and click `Install`.
4. Refresh SillyTavern.
